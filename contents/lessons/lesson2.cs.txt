Lesson: Variables and Data Types in C#

In this lesson, we’ll explore how to declare and use variables in C#.

Variables are used to store information. In C#, you must declare a **type** when creating a variable.

Common types:
- `int` – whole numbers
- `double` – decimal numbers
- `string` – text
- `bool` – true/false

Let’s look at an example:

```cs
int age = 25;
string name = "Maria";
bool isStudent = true;
double height = 1.68;

// Return all variable values in a single string
"Name: " + name +
"\nAge: " + age +
"\nHeight: " + height +
"\nIs Student: " + isStudent
```

When you press ▶️ Run Example, the output box will show all values.
